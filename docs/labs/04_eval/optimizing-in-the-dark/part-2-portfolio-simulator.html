How to use this simulator

1. Click "Generate 10 Agents" to create agents with different E[V] distributions.
2. Panel A sees only point estimates ‚Äî deploy if E[V] > 0.
3. Panel B sees uncertainty ranges ‚Äî deploy only if P(E[V] > 0) > threshold (default 90%).
4. Use "Reduce Uncertainty" to simulate better measurement on held agents.
5. Click "Reveal Truth" to see actual values and outcomes. Try adding bias to see its effect.
