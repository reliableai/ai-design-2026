Course Overview

This course teaches you how to design, build, and evaluate large-scale AI systems. You'll start by building a simple AI-powered application, then dive deep into evaluation methodologies, and finally apply everything in a hands-on project.

Three Phases

  • Phase 1 Lessons 1-3: Building a simple AI app (agents, tools, memory, MCP-style connectors)
  • Phase 2 Lessons 4-10: Evaluation of AI systems (core theory + hands-on)
  • Phase 3 Lessons 11-24: Project-driven work (reviews, meetings, presentations)
