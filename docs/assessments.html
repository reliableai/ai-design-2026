Assessments

Two assessments: midterm exam (40%) and course project (60%)

Grading Breakdown

Assessment Weight
Midterm Exam (Individual) 40%
Course Project (Teams of 1-2) 60%

Midterm Exam

Written Exam - Week 7, Lesson 13

Duration: 90-120 minutes Format: Individual

Topics Covered

The exam covers all material from Weeks 1-6:

Phase 1 Topics (Weeks 1-2)

  • Software 3.0 concepts and paradigms
  • AI agents: design and implementation
  • Tools and tool integration
  • Memory systems for AI applications
  • MCP-style connectors
  • Voice interfaces (optional material)

Phase 2 Topics (Weeks 3-6)

  • Evaluation fundamentals
  • Metrics and rating schemes
  • Golden dataset design and refinement
  • LLM-as-judge approaches
  • Evaluation harness implementation
  • Experimental design and ablations
  • Error analysis and uncertainty quantification
  • AI system architectures

Exam Format

The exam will include a mix of conceptual questions, problem-solving exercises, and short design scenarios. You should be prepared to:

  • Explain key concepts and their trade-offs
  • Analyze evaluation scenarios and propose appropriate metrics
  • Design golden datasets for given use cases
  • Interpret experimental results and identify sources of uncertainty

Course Project

Build & Evaluate an AI System - Weeks 6-12

Team Size: 1-2 students Duration: 7 weeks

The course project is your opportunity to apply everything you've learned. You'll design, build, and rigorously evaluate a large-scale AI system.

Track Options

  • Track A (Class System): Work on a shared class project with defined scope
  • Track B (Capstone): Propose your own project idea
